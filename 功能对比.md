# 蜜蜂记账 - 功能对比报告

## 原始 Flutter 项目 vs 传统前后端项目

---

## 一、原始 Flutter 项目完整功能

### 1.1 核心数据模型

| 模块 | 数据表 | 功能描述 |
|------|--------|----------|
| 账本 | `ledgers` | 多账本管理，支持个人/共享账本 |
| 账户 | `accounts` | 多账户管理（现金、银行卡、信用卡等） |
| 分类 | `categories` | 二级分类系统，支持自定义图标 |
| 交易 | `transactions` | 收支/转账记录，支持标签、附件 |
| 周期交易 | `recurring_transactions` | 周期性自动记账（日/周/月/年） |
| 标签 | `tags` | 自定义标签分类 |
| 预算 | `budgets` | 总预算/分类预算管理 |
| 附件 | `transaction_attachments` | 交易图片附件支持 |
| AI对话 | `conversations`, `messages` | AI记账助手对话记录 |
| 云同步 | `local_changes`, `sync_state` | 增量同步基础设施 |

### 1.2 特色功能

#### 🔴 **AI 智能记账**
- AI 对话助手（基于 GLM-4）
- OCR 图片识别（本地+云端双引擎）
- 语音记账
- 截图自动记账（Android无障碍服务/iOS快捷指令）
- 智能解析金额、商户、分类

#### 🔵 **云同步方案**
- **BeeCount Cloud** (自建+实时同步)
- **iCloud** (iOS原生)
- **Supabase** (免费)
- **WebDAV** (NAS支持)
- **S3协议** (Cloudflare R2/AWS S3/MinIO)

#### 🟢 **数据管理**
- CSV 数据导入/导出
- 配置导入/导出 (YAML)
- 分类迁移工具
- 多语言支持 (简中/繁中/英文)

#### 🟡 **个性化**
- 暗黑模式
- 主题色彩自定义
- 桌面小组件
- 应用锁定

#### 🟣 **统计分析**
- 月度报表
- 分类排行
- 趋势图表

- 语言: Go 1.23+

- ORM: GORM

**前端**
- 框架: React 18
- UI库: Material UI 5
- 路由: React Router 6
- 构建工具: Vite 5
- HTTP客户端: Axios
