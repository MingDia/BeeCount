# BeeCount 用户手册

## 1. 产品介绍
BeeCount是一款智能记账应用，帮助用户轻松管理个人财务，通过AI智能分类提升记账效率。

### 1.1 核心功能
- **智能记账**：AI自动分类，学习用户记账习惯
- **财务分析**：收支趋势、分类分析、预算管理
- **多账户管理**：支持现金、银行卡、信用卡等多种账户类型
- **数据安全**：本地存储，支持数据导出备份
- **跨设备同步**：多端数据同步，随时查看财务状况

## 2. 快速开始
### 2.1 注册和登录
1. **注册账户**
   - 打开BeeCount应用
   - 点击「注册」按钮
   - 输入邮箱、密码、确认密码
   - 点击「注册」完成

2. **登录账户**
   - 打开BeeCount应用
   - 输入邮箱和密码
   - 点击「登录」按钮

### 2.2 创建第一个账本
1. 登录后，点击「创建账本」
2. 输入账本名称（如「个人账本」）
3. 选择货币类型（默认CNY）
4. 点击「创建」

### 2.3 添加第一个交易
1. 在首页点击「快速添加」按钮
2. 选择交易类型（收入/支出/转账）
3. 输入金额
4. 选择分类（系统会智能推荐）
5. 选择账户
6. 输入备注（可选）
7. 点击「保存」

## 3. 功能使用指南
### 3.1 首页
- **财务概览**：显示总收入、支出、余额
- **最近交易**：展示最近10条交易记录
- **快速添加**：一键添加新交易
- **账户卡片**：显示主要账户余额

### 3.2 交易管理
- **交易列表**：查看所有交易记录
- **筛选功能**：按日期、分类、账户筛选
- **搜索功能**：通过关键词搜索交易
- **批量操作**：批量编辑、删除交易

### 3.3 账户管理
- **添加账户**：点击「+」按钮添加新账户
- **编辑账户**：点击账户卡片进入编辑页面
- **账户类型**：现金、银行卡、信用卡、电子钱包、其他
- **初始余额**：设置账户初始余额

### 3.4 分类管理
- **添加分类**：点击「+」按钮添加新分类
- **编辑分类**：点击分类进入编辑页面
- **分类图标**：选择分类图标和颜色
- **分类层级**：支持多级分类

### 3.5 统计分析
- **收支趋势**：图表展示收支变化趋势
- **分类分析**：饼图展示不同分类的支出占比
- **预算管理**：设置每月预算，查看预算使用情况
- **自定义时间**：选择不同时间范围查看数据

### 3.6 AI助手
- **智能分类**：基于交易描述自动分类
- **交易分析**：分析用户消费模式，提供省钱建议
- **预算建议**：根据历史消费数据，智能推荐预算设置
- **学习模式**：学习用户的分类习惯，提高分类准确性

### 3.7 设置
- **个人信息**：修改用户名、邮箱、密码
- **主题设置**：切换浅色/深色模式
- **数据备份**：导出和导入记账数据
- **通知设置**：设置提醒和通知
- **关于我们**：查看版本信息和隐私政策

## 4. 高级功能
### 4.1 周期交易
- **创建周期交易**：设置定期自动生成的交易
- **频率设置**：每天、每周、每月、每年
- **开始和结束日期**：设置周期交易的有效期
- **自动执行**：系统自动生成交易记录

### 4.2 提醒功能
- **交易提醒**：设置交易到期提醒
- **预算提醒**：设置预算使用达到阈值提醒
- **周期交易提醒**：设置周期交易生成提醒

### 4.3 数据导入导出
- **导出数据**：导出为CSV、Excel格式
- **导入数据**：从CSV、Excel文件导入
- **数据迁移**：从其他记账应用迁移数据

### 4.4 高级统计
- **收支对比**：与上月、去年同期对比
- **消费习惯分析**：分析消费模式和习惯
- **财务健康评估**：评估个人财务状况

## 5. 常见问题
### 5.1 登录问题
- **忘记密码**：点击「忘记密码」，通过邮箱重置
- **账户锁定**：多次登录失败后账户会被临时锁定，30分钟后自动解锁

### 5.2 数据问题
- **数据丢失**：使用数据备份恢复
- **同步失败**：检查网络连接，重新同步

### 5.3 功能问题
- **智能分类错误**：手动调整分类，系统会学习改进
- **统计数据不准确**：检查交易数据是否完整

### 5.4 性能问题
- **应用卡顿**：清理缓存，重启应用
- **加载缓慢**：检查网络连接，优化设备性能

- 重启设备
